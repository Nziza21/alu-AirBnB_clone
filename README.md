# AirBnB Clone

## Description

The AirBnB Clone project is a full-stack web application that recreates the core functionality of the AirBnB platform. The goal of the project is to understand how a web application is built from the ground up, including the command interpreter, object storage, database management, API development, and front-end integration.

This first stage focuses on building a command-line interpreter that manages AirBnB objects.

## Command Interpreter

The command interpreter allows users to create, update, delete, and retrieve objects through a command-line interface.

### How to Start It

Clone the repository:

```bash
git clone https://github.com/Nziza21/alu-AirBnB_clone.git
cd alu-AirBnB_clone
```

Run the interpreter:

```bash
./console.py
```

or

```bash
python3 console.py
```

### How to Use It

When started, the interpreter displays the prompt:

```bash
(hbnb)
```

Available commands include:

* create
* show
* destroy
* all
* update
* quit

### Examples

Start the console:

```bash
$ ./console.py
(hbnb)
```

Exit the console:

```bash
(hbnb) quit
$
```

Display help:

```bash
(hbnb) help
```
